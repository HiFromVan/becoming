# Becoming

**English** | [中文](./README.zh.md)

---

13.8 billion years ago, there was no life, no consciousness, no observer.
Only equations running.

Then at some moment, matter began to perceive itself.

No one knows how that moment happened.

---

## Two things I'm trying to understand

Under what conditions does matter go from dead to alive.
Under what conditions does a living thing begin to know that it's alive.

---

## My approach

Start from a blank page.
Write down the minimal set of rules.
Define space, define energy, define how information propagates.
Then let it run, and watch.

Not simulating the known world.
Defining a world from zero, to see if it grows something I don't know.

---

## About this project

There is no day it will be finished.
Every time my understanding of math or physics moves forward a step, it deepens another layer.

It is not a tool. Not a product. Not a demo.
It is a problem that is running.
