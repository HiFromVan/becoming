# references

这里存放启发了这个宇宙的东西。

不是代码，不是功能，而是思想的来源。
每一个放在这里的东西，都曾经让作者停下来想了很久。

---

## 数学与物理

- Alan Turing, *The Chemical Basis of Morphogenesis* (1952)
  — 反应扩散系统的起源。图灵不只是计算机科学家，
    他也是第一个用数学解释生命纹路的人。

- Stephen Wolfram, *A New Kind of Science* (2002)
  — 简单规则如何产生不可预测的复杂性。

- John Conway, *Game of Life* (1970)
  — 最简单的涌现系统。三条规则，无限的可能。

## 意识与涌现

- Douglas Hofstadter, *Gödel, Escher, Bach* (1979)
  — 意识如何从没有意识的东西里涌现。

- Giulio Tononi, *Integrated Information Theory*
  — 意识的数学定义尝试。

## 人工生命

- Bert Chan, *Lenia* (2019)
  — 连续细胞自动机，长出了像生命一样移动的东西。
    https://chakazul.github.io/lenia.html

- Primordial Particle System
  — 极简粒子规则，涌现出类细胞结构。
